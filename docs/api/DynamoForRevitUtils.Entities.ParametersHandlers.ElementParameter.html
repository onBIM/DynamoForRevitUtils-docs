    Class ElementParameter

    Represents a parameter of a Revit element within the Dynamo for Revit utilities.

    Inheritance
    object
    ElementParameter
    Inherited Members

    Namespace: DynamoForRevitUtils.Entities.ParametersHandlers
    Assembly: DynamoForRevitUtils.dll
    Syntax
    public class ElementParameter
    Remarks

    This class acts as a wrapper around Revit's Autodesk.Revit.DB.Parameter object and provides access to the parameter's metadata, value, and additional contextual information necessary in the Dynamo for Revit environment.
