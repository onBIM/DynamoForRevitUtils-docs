    GetProtoSolid(Wall)

    Get the protosolid of a Wall.

    Declaration
    public static Solid GetProtoSolid(this Wall wall)
    Parameters
    Type Name Description
    Wall wall

    The Wall to get the protosolid from.
    Returns
    Type Description
    Solid

    The protosolid of the Wall.
    Remarks

    This method uses a different approach from ToProtoType() method. It uses the Wall's physical location and dimensions to create a Autodesk.DesignScript.Geometry.Solid.

    The returned solid does not contains openings, it is a simple extrusion of the Wall's location curve. Use this method to analyze the Wall's geometry without considering its openings.
