    Class ElementParameterAccessor

    Class used to access properties from a ElementWrapper using ElementParameter

    Inheritance

    Inherited Members

    Namespace: DynamoForRevitUtils.Entities.ParametersHandlers
    Assembly: DynamoForRevitUtils.dll
    Syntax
    public class ElementParameterAccessor
    Remarks

    ElementWrapper must have properties with AssociatedElementParameterAttribute to map the property to the correspondent ElementParameter.

    Examples
    [AssociatedParameterPropertyName(nameof(ViewCommonParameters.DetailNumber))]
public string DetailNumber {
    get => ViewWrapper.ElementParameterAccessor.GetStringValue();
    set => ViewWrapper.ElementParameterAccessor.SetValue(value);
}
