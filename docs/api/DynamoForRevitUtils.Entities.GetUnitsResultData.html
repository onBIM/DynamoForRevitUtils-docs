    Remarks

    Depending on the context some errors might occur. In those cases, returned value could be a Autodesk.Revit.DB.SpecTypeId, instead the desired Autodesk.Revit.DB.UnitTypeId.

    Autodesk.Revit.DB.UnitTypeId represents: meters, square meters, cubic meters etc.

    Autodesk.Revit.DB.SpecTypeId represents: length, area, volume etc.
